Ayuda Controller

El controlador es el proceso que se encarga de gestionar la petición. El "controller" es una clase formada por tantos métodos como procesos necesitemos gestionar. Cuando se ejecuta el controlador se incia en el método New(). Alla si es necesario podremos gestionar el tema de acceso a este controlador y a los roles si es necesario. Los parametros del comando controller son los siguientes
  • controller new [NameController]

    • Cuando creemos con el DS un controlador, el sistema nos crea un plantilla por defecto que podremos gestionar a nuestras necesidades. Podremos indicar carpetas donde se ubicará el controllador, ex: logistica/usuarios

  • controller new [NameController] [NameTemplate]

    • Se creará un controller a partir de la plantilla [NameTemplate]. Podemos tener definidas tantas plantillas como queramos para nuestros desarrollos.

  • controller renew [NameController]

    • En el caso de existir el controller rescribira el fichero

  • controller list

    • Lista el contenido de controllers de nuestro proyecto

  • controller del [NameController]

    • Elimina el controller [NameController]

  • controller show [NameController]

    • Muestra el contenido del controller [NameController]
