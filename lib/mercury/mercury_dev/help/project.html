Ayuda Proyecto

Este proceso te permite crear un proyecto nuevo a partir de un template almacenado. A medida que vamos creando nuestros patrones los podemos reutlizar si los hemos almacenado previamente. Los parametros del comando project son los siguientes
  • project new [NameProject]

    • Crearemos un proyecto nuevo en el directorio [NameProject]. Por defecto usará el template default

  • project new [NameController] [NameTemplate]

    • Se creará un proeycto nuevo a partir del template [NameTemplate]. Podemos tener definidas tantos templates como queramos.

  • project renew [NameProject]

    • En el caso de existir el proyecto lo rescribira

  • project list

    • Lista el contenido de proyectos
